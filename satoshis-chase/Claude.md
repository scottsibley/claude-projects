# Satoshi's Chase

A Pac-Man style browser game for shamory.com, themed around
Bitcoin and the SHAmory brand.

## What this is
A self-contained arcade game playable in any browser.
When complete it will be uploaded directly to SiteGround
and live at shamory.com/satoshis-chase/

## Tech rules
- Everything lives in index.html
- Images and sounds go in the assets/ subfolder
- No external libraries, no CDN links, no dependencies
- Must work by simply opening index.html in a browser
- No iframes — this is a standalone hosted page

## How it will be deployed
Upload the satoshis-chase/ folder via SiteGround File Manager
into the shamory.com public_html directory.

## Definition of done
- Game is playable with arrow keys or WASD
- Has a score counter and basic win/lose states
- Bitcoin and SHAmory themed visuals
- Loads fast and works in desktop browsers
- Ready to upload to SiteGround as-is
