# SHAmory

SHAmory is an ecommerce company. The main website is shamory.com.

## Platform stack
- WordPress (CMS)
- WooCommerce (ecommerce and product management)
- SiteGround (hosting)
- Stripe (card payments)
- OpenNode (bitcoin payments)

## What we build here
Physical and digital projects that support the SHAmory brand and website.
These include games, books, landing pages, tools, and other web experiences for kids.

## Brand tone
Friendly, approachable, and fun. SHAmory customers are the audience.

## Technical defaults for all projects
- Plain HTML, CSS, and JavaScript unless told otherwise
- No heavy frameworks — keep things lightweight and fast
- All files must be deployable directly to SiteGround hosting
- No WordPress plugins required for any project
- No iframes — files are uploaded directly to the server

## Deployment target
SiteGround hosting for shamory.com.
Files are uploaded via SiteGround File Manager in cPanel or FTP.
A project lives at shamory.com/project-name/ once uploaded.
